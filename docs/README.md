# Aethra AI — Premium AI Landing Page

A premium, dark-themed AI website about **Aethra** — the next-generation AI agent platform. Built as a static site with pure HTML, CSS, and JS.

## Features

- Dark, premium design with gradient accents
- Neural sphere hero animation
- Feature showcase with 6 cards
- Model showcase with benchmark scores
- Smooth scroll and intersection-based entrance animations
- **Hidden easter egg**: reveals Hatcher Protocol panels

## Easter Egg 🥚

Click **5 times** anywhere on the page, then press **Shift+A**. The main site will split in half (left slides left, right slides right) and the Hatcher Protocol overlay slides in from both sides.

- Click the overlay background or press **ESC** to close and restore the site.
- Click counter resets after 3 seconds of inactivity.
